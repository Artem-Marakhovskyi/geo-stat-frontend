GeoStat application

Service which provides ability to track userâ€™s location for a certain period and then display it on the map. Application consists of two parts: mobile app and web site. User installs the mobile application on his phone and it tracks his location periodically and sends it to the database. In mobile app or on the web site, user can see on the map where he has been most often for a certain period. Also user can create a group, invite his friends to it and watch the most visited places for all group members on the map.
